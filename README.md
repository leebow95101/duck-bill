# 记账鸭 · `ledger-mini`

一个基于 `uni-app + Vue 3 + Pinia + uView Plus` 的简约风中文记账小程序，主打轻量记账、主题统一、页面简洁，以及适配微信小程序的使用体验。

## 项目特性

- 中文极简记账 UI，主色支持全局统一切换
- 自定义底部 TabBar，中间凸起按钮快速记账
- 首页、账单、统计、我的 四大主页面
- 半屏记账弹层，支持金额、分类、账户、备注、日期选择
- Pinia 模块化状态管理
- 本地存储持久化，无需后端即可体验完整功能
- 微信小程序环境支持真实用户资料授权
- 统计页接入 `uCharts`，当前展示分类占比

## 项目结构

```text
src/
├── components/      通用组件，如自定义导航、TabBar、记账弹层、图表卡片
├── constants/       主题、导航、默认数据等常量
├── pages/           页面文件
├── stores/          Pinia 状态模块
├── styles/          全局样式与 uView 样式入口
├── types/           TypeScript 类型定义
├── utils/           格式化、存储、记账计算等工具函数
├── App.vue          全局样式与应用入口
├── main.ts          挂载 Pinia / uView Plus
└── pages.json       页面路由与导航配置
```

## 状态管理

当前状态按领域拆分为多个 Store：

- `stores/ledger.ts`
  - 核心账单、预算、固定收支、用户资料、页面状态
- `stores/category.ts`
  - 分类管理
- `stores/account.ts`
  - 账户管理与余额同步
- `stores/theme.ts`
  - 全局主题切换与主题变量

## 微信小程序说明

### 1. 微信授权

`我的` 页面中的“微信授权”按钮，已接入微信小程序环境下的真实 `getUserProfile` 授权。

授权成功后会保存：

- 微信昵称
- 微信头像
- 头像首字兜底文本

### 2. AppID

如果你要在微信开发者工具里正常调试授权、登录、接口能力，建议在 `manifest.json` 中配置自己的小程序 `AppID`。

如果未配置真实 `AppID`，部分微信能力在开发者工具中可能受限。

## 主题配置

项目支持统一主题色管理，主题相关配置集中在：

- `src/constants/theme.ts`
- `src/stores/theme.ts`

修改主题配置后，页面会通过 CSS 变量同步更新。

## 当前实现说明

- 统计页当前保留的是分类占比图，不包含收支趋势图
- 所有页面已统一为自定义导航风格
- 账单与资料数据默认保存在本地，不依赖后端服务
- 微信授权仅在 `MP-WEIXIN` 环境下可用，其他平台会给出提示
